\# Soluciones Tecnológicas del Futuro



Plataforma automatizada de despliegue y monitoreo en AWS con prácticas DevOps.



\## Estructura del proyecto

\- /scripts          → Scripts Bash de automatización

\- /python           → Scripts Python con Boto3

\- /docker           → Dockerfile y docker-compose

\- /cloudformation   → Plantillas IaC en YAML

\- /.github/workflows → Pipelines CI/CD



\## Ramas

\- `main`    → Producción (protegida)

\- `develop` → Desarrollo activo



\## Tecnologías

\- AWS (EC2, S3, CloudFormation, CodePipeline)

\- Docker + Nginx/Flask

\- Python + Boto3

\- GitHub Actions CI/CD

\- Terraform / Ansible
